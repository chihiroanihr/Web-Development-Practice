# Packages Installed

- **tailwindcss**
- **postcss**
- **autoprefixer**<br />
  https://tailwindcss.com/<br />
  `npm install -D tailwindcss postcss autoprefixer`

- **react-icons**: https://react-icons.github.io/react-icons<br />
  `npm i react-icons`
